Tools for tracking/incorporating interactive label image production
-------------------------------------------------------------------
These python scripts reduce Renewal Master Content List CSVs to rows and columns containing information relating to the status of images required for the backgrounds.
To use:
1. Get a master content list (eg. 4.MM_Renewal_Master Content List.xls)
2. Convert it to CSV (Google Docs works)
3. Place it in this directory (but don't check it in to version control)
4. Run the associated python script (MM, FSF, MI or MW).
5. Browse the xx_out.csv for the status (and locations, if we're lucky) of the images required.
